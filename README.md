# ElecCal landing

Landing page estatica de captacion para ElecCal, servicio tecnico de electrodomesticos en Ciudad Rodrigo y comarca.

## Contenido

- `index.html`: pagina principal con estilos y formulario de contacto.
- `assets/`: imagenes optimizadas para la landing.

Los formularios y CTAs apuntan a `victorcrodrigo@gmail.com`.
